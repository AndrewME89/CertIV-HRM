# Assessment 1 - Short Answer Questions

Completion requirements

**To do:** Receive a grade 

**To do:** Receive a passing grade

### Student Instructions

This is assessment 1 of 2 assessments for BSBHRM417 [[Support human resources functions and processes]].

This assessment requires you to answer nine (9) short answer questions to test your knowledge required of this unit.

To be assessed as competent, you must complete all tasks in the spaces required.

You are required to download your assessment by clicking on the assessment document icon below (see Let's begin). Before commencing your work, you must update your Student name and Student number in the footer from page 2 onwards. Once the assessment is completed, PDF and upload it for submission (see Submission instructions).

#### Files for Submission

Submit the assessment document with all tasks completed in the spaces provided.

#### Submission Instructions

Please save all your Word documents as PDF files before submitting.

#### Important

1. **Word documents will not be accepted.**
2. **Student name and student number must be captured in the footer from page 2 onwards.**

_Most modern web browsers can open and display a PDF file. If you have an older operating system, however, you may need a PDF reader installed on your device such as the Acrobat Reader, available from [Adobe](https://acrobat.adobe.com/au/en/acrobat/pdf-reader.html)._

##### Windows: Word 2013 and newer

Choose File > Export > Create PDF/XPS.

##### Windows: Word 2010

1. Click the **File** tab
2. Click **Save As**
    - To see the **Save As** dialog box in Word 2013 and Word 2016, you have to **choose a location** and **folder**
3. In the **File Name** box, enter a name for the file, if you haven't already
4. In the **Save as** type list, click **PDF** (*.pdf).

- If you want the file to open in the selected format after saving, select the Open file after publishing check box.
- If the document requires high print quality, click _Standard_ (publishing online and printing).
- If the file size is more important than print quality, click _Minimum size_ (publishing online).

6. Click **Options** to set the page to be printed, to choose whether markup should be printed, and to select output options. Click **OK** when finished.
7. Click **Save**.

##### macOS: Office for Mac

To save your file as a PDF in Office for Mac follow these easy steps:

1. Click the **File**
2. Click **Save As**
3. Click **File Format** towards the bottom of the window
4. Select **PDF** from the list of available file formats
5. Give your file a **name**, if it doesn't already have one, then click **Export**

For more detailed instructions refer to [Microsoft Support](https://support.microsoft.com/en-us/office/save-or-convert-to-pdf-or-xps-in-project-desktop-d85416c5-7d77-4fd6-a216-6f4bf7c7c110#ID0EAADAAA=macOS).
