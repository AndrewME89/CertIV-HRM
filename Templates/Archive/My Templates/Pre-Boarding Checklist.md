## HR Actions

| Task | Owner | Due Date | Done |
|---|---|---|---|
| Employment contract / letter of engagement sent and returned signed | HR | Before start | <input type="checkbox" unchecked> |

| Fair Work Information Statement provided | HR | Before start | <input type="checkbox" unchecked> |
| Modern Award / EBA information provided (if applicable) | HR | Before start | <input type="checkbox" unchecked> |
| Bank account details collected for payroll | HR | Before start | <input type="checkbox" unchecked> |
| Employee added to HRIS | HR | Before start | <input type="checkbox" unchecked> |
| Payroll set up completed | Payroll | Before first pay cycle | <input type="checkbox" unchecked> |
| Personnel file created | HR | Before start | <input type="checkbox" unchecked> |
| Pre-employment checks completed (police check, WWCC, etc.) | HR | Before start | <input type="checkbox" unchecked> |
| Emergency contact details collected | HR | Before start | <input type="checkbox" unchecked> |
| Any workplace adjustment requirements identified and actioned | HR / Manager | Before start | <input type="checkbox" unchecked> |
| Employee added to relevant mailing lists / org charts | HR | Before start | <input type="checkbox" unchecked> |
