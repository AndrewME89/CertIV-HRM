# Inclusive Job Ad Language Checklist

> Use this checklist before publishing any job advertisement. Based on Australian anti-discrimination legislation and best-practice inclusive recruitment guidance.

---

## Section 1: Language Tone & Framing

| Check | ✓ | Notes |
|---|---|---|
| The ad uses plain, active language | | |
| The role is described in terms of outcomes, not personality archetypes | | |
| The ad avoids listing "culture fit" as a criterion | | |
| The ad does not use unnecessarily gendered nouns (e.g., "salesman" → "sales rep"; "manpower" → "workforce") | | |
| The ad avoids sports or military metaphors that may alienate some readers (e.g., "ninja," "rockstar," "warrior") | | |
| The ad does not use phrases implying a particular demographic is preferred (e.g., "young," "recent graduate," "mature") | | |

---

## Section 2: Requirements Audit

| Check | ✓ | Notes |
|---|---|---|
| Every listed requirement is genuinely necessary for the role | | |
| Qualifications required are mandated by law or genuinely essential (not merely preferred) | | |
| Years of experience requirements have been scrutinised for age discrimination risk | | |
| Physical requirements are listed only where operationally essential and cannot be reasonably adjusted | | |
| Language/communication requirements reflect only what the job actually demands | | |
| Licences/certifications listed are current and valid for the jurisdiction | | |

---

## Section 3: Diversity & Inclusion

| Check | ✓ | Notes |
|---|---|---|
| An equal opportunity employer statement is included | | |
| The statement specifically encourages Aboriginal and Torres Strait Islander applicants | | |
| The statement encourages people with disability to apply | | |
| The statement encourages CALD applicants | | |
| The statement encourages LGBTIQ+ applicants | | |
| The statement encourages applicants of all ages | | |
| Reasonable adjustments are offered for people with disability during the recruitment process | | |
| Flexible working options are mentioned (if available) | | |

---

## Section 4: Transparency & Compliance

| Check | ✓ | Notes |
|---|---|---|
| Salary range is disclosed (best practice; required in some states/sectors) | | |
| Employment type is clearly stated | | |
| Work location and flexibility are clearly stated | | |
| Pre-employment check requirements are disclosed upfront | | |
| Right-to-work requirement is stated | | |
| Visa sponsorship position is stated (yes or no) | | |
| Application closing date is included | | |
| Contact person for enquiries is named | | |
| The ad complies with the applicable Award/EBA classification | | |

---

## Section 5: Platform-Specific Checks

| Platform | Check | ✓ |

## Flagged Terms Reference

### Avoid → Use Instead

| Avoid | Use Instead |
|---|---|
| Salesman / saleswoman | Sales representative / sales professional |
| Manpower | Workforce / staffing |
| Guys | Team / everyone |
| Digital native | Proficient with [specific tools] |
| Young and energetic | Enthusiastic / motivated |
| Mature and experienced | Experienced |
| Recent graduate | Entry-level / early career |
| Ninja / rockstar / guru | Skilled / expert / specialist |
| Work hard, play hard | High-performance / collaborative |
| Must be physically fit | Must be able to [specific task] |
| Native English speaker | Fluent English communication skills |
| Clean-shaven | Professional appearance per uniform policy |
