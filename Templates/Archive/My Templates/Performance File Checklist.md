## Section 4: Document Quality Check

For each performance or disciplinary document, confirm:

| Quality Check | ✓ |
|---|---|
| Specific incidents/dates are recorded (not vague generalisations) | <input type="checkbox" unchecked> |
| Documents are signed by both employee and manager | <input type="checkbox" unchecked> |
| Employee's opportunity to respond is documented | <input type="checkbox" unchecked> |
| Support offered to employee is documented | <input type="checkbox" unchecked> |
| Outcomes and consequences are clearly stated | <input type="checkbox" unchecked> |
| HR involvement/sign-off is recorded where required | <input type="checkbox" unchecked> |
| No protected attribute information (health, family, etc.) is embedded in performance records without legal basis | <input type="checkbox" unchecked> |
| Access to file is restricted to authorised personnel | <input type="checkbox" unchecked> |
