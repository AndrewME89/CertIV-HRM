# Natural Justice Checklist

## The Three Elements of Natural Justice

### 1. The Right to Know (Disclosure)

| Check | ✓ | Notes |
|---|---|---|
| The employee has been given a clear and specific account of the allegations or concerns against them | <input type="checkbox" unchecked> | |
| The allegations were provided in writing (preferred) with sufficient detail to allow a meaningful response | <input type="checkbox" unchecked> | |
| The employee was told what policies, standards, or obligations are alleged to have been breached | <input type="checkbox" unchecked> | |
| The employee was given the evidence that supports the allegations (where appropriate and legally permissible) | <input type="checkbox" unchecked> | |
| The employee was given adequate time to consider the information before being required to respond | <input type="checkbox" unchecked> | |

---

### 2. The Right to Respond (Opportunity to be Heard)

| Check | ✓ | Notes |
|---|---|---|
| The employee was given a genuine opportunity to respond — in writing, verbally, or both | <input type="checkbox" unchecked> | |
| The employee was offered the right to bring a support person to any meeting | <input type="checkbox" unchecked> | |
| The employee's response was genuinely considered before any decision was made | <input type="checkbox" unchecked> | |
| If new information emerged from the employee's response, this was properly investigated | <input type="checkbox" unchecked> | |
| The employee was not asked to respond on the spot without adequate time to prepare | <input type="checkbox" unchecked> | |

---

### 3. The Decision-Maker Must Be Unbiased (Absence of Bias)

| Check | ✓ | Notes |
|---|---|---|
| The decision-maker was not directly involved in the incident (or if so, this is documented and justified) | <input type="checkbox" unchecked> | |
| No prior adverse decision about this employee has influenced the outcome | <input type="checkbox" unchecked> | |
| The decision-maker has not pre-judged the matter (e.g., verbal comments about outcome before process concluded) | <input type="checkbox" unchecked> | |
| The process has been consistent with how similar matters have been handled for other employees | <input type="checkbox" unchecked> | |
